## Why Gumdrop Studio?

<CardGrid stagger>
  <Card title="🎨 Dual-Layer Canvas" icon="pencil">
    Draw crisp pixel art on the pixel layer, then add smooth anti-aliased vector
    accents on the overlay — perfect for whiskers, eyelashes, highlights, and
    fine details.
  </Card>
  <Card title="🛠️ 8 Drawing Tools" icon="setting">
    Pixel Stamp, Pencil, Eraser, Line, Rectangle, Circle, Bezier Curve, and
    Accent Pen. Everything you need for great pixel art.
  </Card>
  <Card title="🌈 HSV Color Wheel" icon="sun">
    Full-spectrum color picker with alpha transparency support, hex input, and
    color persistence across sessions.
  </Card>
  <Card title="📤 5 Export Formats" icon="rocket">
    Export as PNG, JPG, SVG, JSON, or a self-contained HTML file — ready for
    web, games, portfolios, or re-editing.
  </Card>
  <Card title="💾 Project Management" icon="open-book">
    Save and load unlimited projects. Auto-save keeps your work safe. Works
    in-browser or with native storage on desktop.
  </Card>
  <Card title="🖥️ Web & Desktop" icon="laptop">
    Use it right in your browser at gumdropstudio.app, or install the native
    Linux desktop app for system menus and keyboard shortcuts.
  </Card>
</CardGrid>
