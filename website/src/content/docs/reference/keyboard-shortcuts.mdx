<Aside type="note">
Keyboard shortcuts are only available in the **desktop app**. The browser version at [gumdropstudio.app](https://gumdropstudio.app/) does not have system-level keyboard shortcuts.
</Aside>

---

## File

| Shortcut       | Action       |
| -------------- | ------------ |
| `Ctrl+N`       | New project  |
| `Ctrl+O`       | Open project |
| `Ctrl+S`       | Save         |
| `Ctrl+Shift+S` | Save As      |

## Export

Use **File → Export** in the menu bar to access all five export formats: PNG, JPG, SVG, JSON, and HTML.

## Edit

| Shortcut       | Action |
| -------------- | ------ |
| `Ctrl+Z`       | Undo   |
| `Ctrl+Shift+Z` | Redo   |

## View

| Shortcut | Action                 |
| -------- | ---------------------- |
| `+`      | Zoom in                |
| `-`      | Zoom out               |
| `Ctrl+G` | Toggle pixel grid      |
| `Ctrl+D` | Toggle dark/light mode |

---

## Menu Bar Reference

The desktop app has a full native system menu bar:

| Menu     | Items                                                          |
| -------- | -------------------------------------------------------------- |
| **File** | New, Open, Save, Save As, Export (PNG/JPG/SVG/JSON/HTML), Quit |
| **Edit** | Undo, Redo                                                     |
| **View** | Zoom In, Zoom Out, Toggle Grid, Toggle Dark Mode               |
| **Help** | About Gumdrop Studio, Visit Website                            |
